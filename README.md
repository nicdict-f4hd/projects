# Basic HTML Website

A multi-page website built using **only HTML**, without any CSS or JavaScript.  
This project is part of the **roadmap.sh Frontend Projects** and focuses on understanding core HTML structure, navigation, and page linking.

## Pages Included

- **Home** – Main landing page
- **Projects** – List of projects
- **Articles** – Articles section (placeholder content)
- **Contact** – Contact information page

## Features

- Multi-page navigation using `<a>` tags
- Semantic HTML structure
- No CSS or JavaScript (HTML only)
- Clean and beginner-friendly layout

## Technologies Used

- HTML5
